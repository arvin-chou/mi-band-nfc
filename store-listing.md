## Full Description (4000 chars max)

Mi Band NFC Manager 讓你輕鬆管理小米手環上的 NFC 卡片。

🔑 主要功能：
• 查看手環上所有 NFC 卡片
• 一鍵切換預設卡片（門禁卡、交通卡、支付卡）
• 自動切換規則 — 設定時間自動切換（上班切門禁、下班切交通卡）
• 快速切換面板 — 首頁直接點選切換

⚡ 技術特色：
• 透過 BLE 直接與手環通訊，不需要小米健康 App
• 支援 Xiaomi Smart Band 8 NFC 版
• Material Design 3 現代 UI
• 完全開源，程式碼公開透明

💡 使用前準備：
1. 需要小米手環 8 NFC 版
2. 需要手環的 Auth Key（可從 Notify for Xiaomi 取得）
3. 手環需已透過小米健康 App 配對並新增 NFC 卡片

📱 支持的手環型號：
• Xiaomi Smart Band 8 NFC

🆓 免費且開源：

贊助開發者後可移除廣告。
原始碼公開於 GitHub (github.com/arvin-chou/mi-band-nfc)。
